# Setup Guide

## 1. Install VEXcode V5
Download: https://www.vexrobotics.com/vexcode/install/v5

## 2. (Optional) Install PROS
For advanced C++ development: https://pros.cs.purdue.edu/v5/getting-started/

## 3. Install GitHub Desktop
Download: https://desktop.github.com/

## 4. Clone this repo
In GitHub Desktop: `File → Clone Repository` → search for this repo

## 5. Connect the Robot Brain
- Use the VEX V5 USB cable
- Open VEXcode → your program should appear in the device list
