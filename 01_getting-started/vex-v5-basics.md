# VEX V5 Basics

## Key Components
- **Brain** — the main computer on the robot
- **Motors** — up to 20 smart motors, each with built-in encoder
- **Controller** — handheld, connects wirelessly or via cable
- **Sensors** — distance, rotation, inertial (IMU), optical, etc.
- **Radio** — for wireless controller connection

## Ports
- 21 smart ports on the Brain (motors + sensors share these)
- 3-wire ports for simpler sensors and pneumatics
