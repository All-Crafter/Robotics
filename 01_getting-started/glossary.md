# Glossary

| Term | Meaning |
|------|---------|
| Auton | Autonomous period (first 15 seconds of a match) |
| Driver control | The 1:45 min period where a human drives |
| PID | Control algorithm to move to a target position accurately |
| Odometry | Tracking the robot's position on the field using encoders |
| IMU | Inertial Measurement Unit — gyroscope + accelerometer |
| Encoder | Sensor that counts motor rotations |
| Strafing | Moving sideways (only possible with holonomic/mecanum drive) |
