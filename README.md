# Robotics — VEX V5

This repository contains everything our school's robotics team has learned over the years.
Built to help the **next generation** get started faster and avoid the mistakes we made.

## What's inside

| Folder | Content |
|--------|---------|
| `01_getting-started/` | Setup guides, basics, workflow |
| `02_code/` | All programs, organized by season |
| `03_mechanics-cad/` | Build principles, CAD files, parts lists |
| `04_competitions/` | Match logs, strategies, results |
| `05_knowledge/` | PID, odometry, sensors, lessons learned |
| `06_resources/` | Useful links and learning materials |
| `07_archive/` | Old stuff we don't want to delete |
