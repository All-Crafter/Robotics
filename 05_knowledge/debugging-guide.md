# Debugging Guide

## Robot not connecting to Brain
- Check USB cable
- Restart VEXcode
- Re-flash firmware via VEXcode

## Auton not running
- Check that auton selector is set
- Verify motor ports match code
- Check battery level (low battery = unpredictable behavior)

## Motors not moving
- Check port numbers
- Check for motor faults in Brain diagnostics
- Check gear cartridge

_Add more as you encounter them._
