# Contributing

## Branches
- `main` — stable, tested code only
- `dev` — active development
- `season-YYYY` — per-season branches

## Commit messages
Keep them short and descriptive:
```
fix: PID tuning for auton
feat: add odometry module
docs: update setup guide
```

## Pull Requests
- At least one other team member must review before merging to main
- Always test on the robot before submitting a PR

## Adding knowledge
If you learn something useful — document it in `05_knowledge/`.
Especially failures. Future you will be grateful.
